# WebhooksHealthCheckStatusStatus

Latest health status of the webhooks system

## Example Usage

```ruby
require "gusto_embedded_client"

value = WebhooksHealthCheckStatusStatus::HEALTHY
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `HEALTHY`   | healthy     |
| `UNHEALTHY` | unhealthy   |
| `UNKNOWN`   | unknown     |

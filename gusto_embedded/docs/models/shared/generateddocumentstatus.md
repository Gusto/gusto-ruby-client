# GeneratedDocumentStatus

Current status of the Generated Document

## Example Usage

```ruby
require "gusto_embedded_client"

value = GeneratedDocumentStatus::PENDING
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `PENDING`   | pending     |
| `STARTED`   | started     |
| `SUCCEEDED` | succeeded   |
| `FAILED`    | failed      |

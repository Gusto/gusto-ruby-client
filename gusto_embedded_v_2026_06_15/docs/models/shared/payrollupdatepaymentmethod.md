# PayrollUpdatePaymentMethod

The employee's compensation payment method. Invalid values will be ignored.

## Example Usage

```ruby
require "gusto_embedded_client_v_2026_06_15"

value = PayrollUpdatePaymentMethod::DIRECT_DEPOSIT
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `DIRECT_DEPOSIT` | Direct Deposit   |
| `CHECK`          | Check            |

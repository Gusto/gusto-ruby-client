# PayrollShowAmountType

The amount type of the deduction for the pay period. Only present for unprocessed payrolls.

## Example Usage

```ruby
require "gusto_embedded_client_v_2026_06_15"

value = PayrollShowAmountType::FIXED
```


## Values

| Name      | Value     |
| --------- | --------- |
| `FIXED`   | fixed     |
| `PERCENT` | percent   |

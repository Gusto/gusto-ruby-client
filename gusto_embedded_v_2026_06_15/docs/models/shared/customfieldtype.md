# CustomFieldType

Input type for the custom field.

## Example Usage

```ruby
require "gusto_embedded_client_v_2026_06_15"

value = CustomFieldType::TEXT
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `TEXT`     | text       |
| `CURRENCY` | currency   |
| `NUMBER`   | number     |
| `DATE`     | date       |
| `RADIO`    | radio      |

# PaymentConfigsUpdateRequestPaymentSpeed

Desired payment speed. 1-day is only applicable to partners that opt in.

## Example Usage

```ruby
require "gusto_embedded_client_v_2026_06_15"

value = PaymentConfigsUpdateRequestPaymentSpeed::ONE_MINUS_DAY
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `ONE_MINUS_DAY`  | 1-day            |
| `TWO_MINUS_DAY`  | 2-day            |
| `FOUR_MINUS_DAY` | 4-day            |

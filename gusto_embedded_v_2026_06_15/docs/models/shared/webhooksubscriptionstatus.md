# WebhookSubscriptionStatus

The status of the webhook subscription.

## Example Usage

```ruby
require "gusto_embedded_client_v_2026_06_15"

value = WebhookSubscriptionStatus::PENDING
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `PENDING`     | pending       |
| `VERIFIED`    | verified      |
| `REMOVED`     | removed       |
| `UNREACHABLE` | unreachable   |

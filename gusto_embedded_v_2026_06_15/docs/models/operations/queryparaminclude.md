# QueryParamInclude

Available options:
- all_benefits: Include all effective dated benefits for each employee instead of only the current benefits.

## Example Usage

```ruby
require "gusto_embedded_client_v_2026_06_15"

value = QueryParamInclude::ALL_BENEFITS
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `ALL_BENEFITS` | all_benefits   |

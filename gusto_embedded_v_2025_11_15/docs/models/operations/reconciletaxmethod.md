# ReconcileTaxMethod

How Gusto will handle taxes already collected.

## Example Usage

```ruby
require "gusto_embedded_client_v_2025_11_15"

value = ReconcileTaxMethod::PAY_TAXES
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `PAY_TAXES`    | pay_taxes      |
| `REFUND_TAXES` | refund_taxes   |
